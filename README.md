# Iris Dashboard — Dash

Interactive logistic-regression dashboard on the classic Iris
dataset, built with **Dash** and Plotly.

Part of a three-framework comparison — the same app implemented in:
[streamlit-iris](https://github.com/jasperchenn/streamlit-iris) / [panel-iris](https://github.com/jasperchenn/panel-iris)

## Run

```bash
pip install -r requirements.txt
python dash_plotly_logistic_reg_csv.py
```

## Entry point

`dash_plotly_logistic_reg_csv.py`
